# Abstract Tier List Maker

This is a React-based Tier List application where you can upload project images, rank them into tiers, and share the results.

## Features

- **Custom Tiers**: S, A, B, D, SKIP, C (matching your request).
- **Drag & Drop**: Smooth dragging between the item bank and tiers, and reordering within tiers.
- **Upload**: Add your own project images/icons.
- **Export**: Download the tier list as an image.
- **Share**: Quickly share your tier list on X (Twitter).

## How to Run

1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Start the development server:
    ```bash
    npm run dev
    ```
3.  Open `http://localhost:5173` in your browser.

## Tech Stack

-   React + Vite
-   Tailwind CSS
-   @dnd-kit (Drag and Drop)
-   html2canvas (Screenshot generation)
